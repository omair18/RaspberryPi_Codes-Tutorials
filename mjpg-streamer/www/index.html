WiFi-BOT

Introduction

The Wi-FiBot is a four wheeled robot toy car which would be operated through the means of Wi-Fi. It would have a camera fitted on it which would enable the user to view the surroundings of the robot. Using the simple up/down/left/right arrow keys on the keyboard, the user would be able to move the robot around. Also, with video, it'd be able to stream audio back to the user end too.
